# Changelog

## [1.0.0] - 2024-06-01

### Added

- Initial release of Enterprise Mega App.
- Tech stack: React 18, TypeScript, Vite, Tailwind CSS.
- Demo data store (localStorage) for users, subscriptions, payments, analytics.
- User management: add, edit, delete users; assign roles; toggle active/inactive.
- Subscription management: add, edit, delete subscriptions; status and plan management.
- Payment simulation: simulate payments with various methods and statuses.
- Analytics: track and display user events and system analytics.
- Role-based access: admin-only and protected routes.
- Responsive UI with Tailwind CSS.
- Accessibility: semantic HTML, keyboard navigation, ARIA attributes.
- Testing: 100% user-centric tests with @testing-library/react and Vitest.
- Environment configuration via `.env` and feature flags.
